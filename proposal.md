# サロン売上予測ツール 導入提案書

---

# 1 これまでの問題

- 毎月の売上予測作成に時間がかかっていた
- スプレッドシートで全て手計算（ヘルプ移動、季節変動、稼働率など）
- 計算ミスのリスクがあった
- スタッフの異動・休職時に計算式の修正が必要だった

---

# 2 既存の流れ

**ステップ1** スタッフのランク・所属店舗をスプシで管理

↓

**ステップ2** 季節係数を確認し、各スタッフの売上を手計算

↓

**ステップ3** ヘルプ（店舗間移動）の減算・加算を手計算

↓

**ステップ4** 産休・休職者の売上を手動で調整

↓

**完了** 店舗ごとに集計し、全店舗の合計を算出

---

# 3 改善内容

### 入力するだけで自動計算！

スタッフ情報・ヘルプ・稼働率を入力するだけで、売上予測が自動計算されるWebツールを導入します。

**メリット①：リアルタイムで売上予測を確認！**

- 年月を選ぶだけで、全店舗の売上予測を即座に表示
- 店舗をクリックするとスタッフ別の内訳も確認可能

**メリット②：ヘルプ計算も自動！**

- ヘルプ先店舗と移動%を入力するだけ
- 所属店舗からの減算、ヘルプ先への加算を自動計算
- 複数回のヘルプにも対応

**メリット③：稼働率で柔軟に対応！**

- 産休・休職は5%刻みで稼働率を設定可能
- 月の途中から休む場合も正確に計算（例：25%稼働）

**メリット④：季節変動も自動適用！**

| 季節 | 対象月 | 係数 |
|------|--------|------|
| 通常期 | 1, 3, 4, 9, 10, 11月 | 100% |
| 閑散期 | 2, 5, 6月 | 98% |
| 繁忙期 | 7, 8月 | 106% |
| 超繁忙期 | 12月 | 110% |

**今回の制作で意識したこと**

1. **誰でも操作できる**（直感的なUI、使い方ガイド付き）
2. **時間削減**（入力するだけで自動計算）
3. **拡張性**（店舗・スタッフ追加にシステム改修不要）

---

# 4 導入・運用手順（クイックスタート）

1. **初期設定（1回だけ）**
   - 店舗を登録
   - スタッフ情報を登録（名前、所属店舗、職種、ランク）

2. **月次の運用**
   - ヘルプがある場合：「ヘルプ入力」画面で登録
   - 休職者がいる場合：「スタッフ管理」画面で稼働率を設定
   - 「売上予測」画面で結果を確認

3. **予測の保存**
   - 「予測を保存」ボタンで履歴に保存
   - 過去の予測は「予測履歴」画面で確認可能

4. **トラブルシューティング**
   - 売上が0の場合：スタッフのランク設定を確認
   - ヘルプが反映されない：年月が正しいか確認
   - 使い方が分からない：「使い方」ページを参照

# 6 機能一覧

| 機能 | 説明 |
|------|------|
| 売上予測 | 全店舗の月間売上予測を自動計算・表示 |
| 店舗管理 | 店舗の追加・編集・削除 |
| スタッフ管理 | スタッフ情報の登録、月別稼働率の設定 |
| ヘルプ入力 | 店舗間の人員移動による売上移動の設定 |
| 予測履歴 | 過去の予測データの保存・閲覧 |
| 福袋対応 | チェックで物販売上2倍計算 |
| 使い方ガイド | 詳細な操作マニュアル |
